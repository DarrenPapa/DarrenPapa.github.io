The new parser is under development!
It is under "py_parser2.py" in "lib.core".
It is available to play with within "dpl_module.py"! 

from dpl import dpl_module
code = "# dpl code here"
executor = dpl_module.parser2.executor(code)
executor.resolve_instructions()
executor.run()
# if youre running the same code once
# then this is not necessary.
executor.reset()

Features supported by the new parser

(preprocessing directives are still the same
they are sepparate from runtime after all)
Old -> New

fn name(...) - same
catch (...) func ... -> catch (...) = func ...
scatch, smcatch - unsupported
if - same
while - same
loop - same
for name in iter - same
for index, name in iter - unsupported
set name = value - same
del name, ... - same
object, new - unsupported
io:println, io:print -> println, print (intrinsics)
io:input -> .input (expression) "set user_input = .input"
dict - unsupported
method - unsupported
exit - unsupported
scatch - unsupported
